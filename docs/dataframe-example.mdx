## Features

The DataFrame component supports:
- Sortable columns (click on column headers)
- Pagination with customizable rows per page
- Responsive design with horizontal scrolling
- Dark mode support
- Column resizing (drag column edges)
- Hover tooltips for long content
- Even/odd row styling
- **Metadata toggle** - Hide/show technical fields with a slider switch
